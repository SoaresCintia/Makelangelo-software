Rapport sur l'ajout de tests dans le projet Makelangelo-software

Par Larry Fotso Guiffo et Cíntia Dalila Soares

Introduction

Ce rapport documente mon travail consistant à ajouter 10 nouveaux tests dans le projet Makelangelo-software.

Tests Ajoutés

Les tests ajoutés se concentrent la classe TransformedImage :

Tests pour la méthode canSampleAt(double x, double y) :

  • testImageInside() : teste que si on donne un point à l'interieur de l'image, la méthode retourne True .
  • testImageOutside() : teste que si on donne un point à l'exterieur de l'image, la méthode retourne False .

Tests pour la méthode sample(double x, double y) :

  • testSampleZero() : teste une branche de la méthode que si on envoie des valeurs enversées (gauche vs droite et haut vs bas) et que si le point à gauche et en bas est à l'exterieur de l'image, on devrait recevoir la valeur 0.
  • testSampleUnchecked() : ... .

Les tests ajoutés se concentrent sur la classe NameThatColor :

Tests pour la méthode find(Long r, Long b , Long g) :

  • findColorTest() : Test si la methode retourne la bonne coleeur quand un code rbg lui est donne en parametre (eg: find(255,255,255) --> white).
  • findColorTest2() : Test si la methode retourne la bonne couleur quand un objet Color lui est passe en parametre(find(new Color(0, 0, 0)) --> black) .
  • findColorTest3() :Test si la methode retourne la bonne couleur quand un objet ColorRGB lui est passe en parametre(new ColorRGB(255, 0, 0)--> red)
  • findColorTest4() :Test si la methode retourne la bonne couleur quand un objet Long lui est passe en parametre(new Long(0xFF0000) --> red)

Tests pour la méthode le constructeur :

  • arrayColorContentTest() : Test si l'initialisation de la l'array list des coleur est bien reussit

Les tests ajoutés se concentrent sur la classe ColorRGB :

Tests pour la méthode set(ColorRGB c) :

  • TestColorRGB() : Vas tester si les champs des red,green et de l'objet qui appel la methode set a bien ete change

Tests pour la méthode sub( , , ..) :

  • TestColorRGBSub() : Test si la methode sub retourne bien un objet qui a des values red,green et blue different de l'objet que sub() prend en parametre.
